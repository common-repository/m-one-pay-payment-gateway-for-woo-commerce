Implement M One Payment gateway your website and start selling with WooCommerce products using this gateway.

== Description ==

This M One Payment gateway plugin will allow you to sell products or services on your website using M One Payment gateway.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/m1pay-gateway` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Woo Commerce->Settings->Payments->M One Gateway and click on manage button to configure the plugin.

== Frequently Asked Questions ==

= My M One Gateway Plugin does not redirect to M One Gateway website =

Check you have entered the correct authentication information into the setting section.

= After payment is done or canceled the M One gateway does not redirect to the correct address on my website =

You have to send the correct URL to the M One support team to set as redirect address. You can pick the URL from plugin setting page.
